# Dashboard Mockup: Sales Performance Optimization Dashboard

## Visual Description

This markdown file describes the layout and components of the Sales Performance Optimization Dashboard built in Power BI.

### Overall Layout

The dashboard follows a multi-page structure optimized for different user personas:
- **Executive Summary Page**: High-level KPIs and top-level metrics
- **Regional Performance Page**: Detailed regional analysis with drill-down capabilities
- **Product Analysis Page**: Product-level insights and trends

### Executive Summary Page Components

1. **Header Section**
   - Dashboard title: "Sales Performance Optimization Dashboard"
   - Last updated timestamp
   - Date range filter (allowing selection of time periods)

2. **KPI Cards (Top Row)**
   - Total Revenue: Large number card with trend indicator (↑/↓ vs. previous period)
   - Variance vs. Target: Percentage card showing overall performance against target
   - Total Units Sold: Count card with comparison to target
   - Average Order Value: Metric card showing AOV trend

3. **Regional Performance Chart (Left Column)**
   - Horizontal bar chart comparing actual revenue vs. sales target by region
   - Color-coded bars (green for above target, orange for below target)
   - Tooltip shows variance percentage on hover

4. **Time Trend Analysis (Right Column)**
   - Line chart showing monthly revenue trends
   - Two lines: Actual Revenue (solid) and Sales Target (dashed)
   - Enables identification of seasonal patterns and trends over time

5. **Top Products Table (Bottom)**
   - Data table showing top 10 products by revenue
   - Columns: Product Name, Revenue, Units Sold, Variance %
   - Sortable columns for interactive exploration

6. **Product Category Breakdown**
   - Pie chart or stacked bar chart showing revenue distribution by category
   - Highlights which categories drive the most revenue

### Visual Design Notes

- **Color Scheme**: Professional blue/green palette with red/orange for variance alerts
- **Typography**: Clear, readable fonts with appropriate sizing hierarchy
- **Interactivity**: All charts are clickable for drill-down and filtering
- **Mobile Responsive**: Layout adapts for tablet/mobile viewing

### Key Features

- Cross-filtering: Selecting a region filters all other visuals on the page
- Drill-through: Click on a region to see detailed product-level analysis
- Export capability: Users can export data to Excel or PDF
- Bookmark navigation: Quick navigation between saved views
