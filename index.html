03 / about

I like fixing the
unglamorous stuff.

I'm based in Monroe, LA. The problems I keep saying yes to look like this: a team still downloading the same reports by hand, a SaaS tool that won't expose an API on the plan you can afford, or an owner who needs one screen to start the day — not another spreadsheet.

That's what pulled me into property management tech at Braud. Buildium's Growth plan has no API, so I wrote Playwright automation. Ownership needed a morning view, so I built the dashboard on top of the same PostgreSQL database the pipeline feeds. Tenants needed a place to browse units, so I shipped braudproperties.com.

Before Braud, I spent four years as an IT contractor across 20+ client environments — hardware, imaging, diagnostics, the work that breaks when nobody's watching. I'm still the person who wants to see the data pipeline and sit with the person who has to use it.
