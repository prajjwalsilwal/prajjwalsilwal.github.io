Overview

Full-stack production website for Braud & Son Property Management in Monroe, LA. Displays live rental listings across 200+ units and 10 properties, with Sanity CMS for content management, real-time listing filters, and direct Buildium integration for online tenant applications.

The Problem

Prospective tenants had no centralized place to browse available units, filter by property or price, or submit applications online — everything routed through phone calls and manual follow-up.

The Solution

A Next.js and React front-end with Sanity CMS for editable content, Tailwind CSS for responsive design, and Buildium integration so listings stay current and applications flow directly into the property management system.

Visit the live site

The production website is live at braudproperties.com — browse listings, filters, and online applications.
