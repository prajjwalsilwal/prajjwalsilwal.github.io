# Project Images

Drop your images here. Supported formats: .jpg, .jpeg, .png, .webp

## Expected filenames for this project:

| Filename           | Used for                              |
|--------------------|---------------------------------------|
| hero.jpg           | Project card thumbnail + case study header |
| architecture.jpg   | Pipeline/system diagram               |
| dashboard.jpg      | Dashboard or output screenshot        |

## Tips:
- hero.jpg: recommended 1200×630px (16:9)
- Other images: 1200px wide minimum
- Keep file sizes under 500KB (use squoosh.app to compress)
- If no image is uploaded, a gradient placeholder renders automatically
